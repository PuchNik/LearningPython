Список записывается:
    cities = ['Moscow', 'LA', 'CH']
    cities[0]

! Список - изменяемый тип данных. Список – динамическая структура данных, который может меняться в процессе
  работы программы.
    full[0] = 3

В списке могут присутствовать любые типы данных:
    full = ['hi', 7, True, 12.3, False, [1, 2, 3], -7]

Создание пустого списка:
    a = []
    или через функцию
    b = list()

! Функции работы со списками:
    len() – определение числа элементов в списке (длина списка);
    max() – для нахождения максимального значения;
    min() – для нахождения минимального значения;
    sum() – для вычисления суммы;
    sorted() – для сортировки коллекции.

Операторы списков:
    + – соединение двух списков в один;
    * – дублирование списка;
    in – проверка вхождения элемента в список;
    del – удаление элемента списка.

! in - позволяет определить вхождение списка:
  lst = ["Москва", 1320, 5.8, True, "Тверь", False]
    1320 in lst
    120 in lst

! del - удаление значения из списка:
  lst = ["Москва", 1320, 5.8, True, "Тверь", False]
    del lst[2]
