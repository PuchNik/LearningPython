1) append - вставить последний элемент в список
2) insert - вставить элемент в определенное место списка ПЕРЕД каким либо элементом
3) remove - можно удалять как по индексу, так и по типу
    1. По индексу: marks.remove(marks[2])
    2. По типу: marks.remove(True)
4) pop - запись может быть с аргументами и без них
    1. С аргументами: по индексу элемента a.pop(3)
    2. Без аргументов: a.pop() - удаление последнего элемента по дефолту
Также можно записать значение удаленного элемента в переменную:
    end = a.pop()
5) clear - очистка полного списка
6) copy - создание копии существующего списка тремя способами:
    1) c = a.copy()
    2) c = a[:]
    3) c = list(a)
7) count - позволяет найти число элементов с указанным значением
8) index - определяет индекс определенного значения:
    a.index(значение, место по-счету в списке)
    a.index(1, 2)
9) reverse - меняет порядок следования элементов на обратный
10) sort - выполняет сортировку значений по возрастанию текущего списка
    Если мы хотим вывести список по:
        1) Возрастанию: a.sort()
        2) Убыванию: a.sort(reverse=True)
    ! sort и sorted: sorted возвращает новый отсортированный список, при этом прежний список остается неизменным.
                     sort применяется именно к текущему списку и при этом ничего не возвращает
