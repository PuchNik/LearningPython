Проверка id у переменной - id(var)
Проверка типа переменной - type(var)

Типы:
    int
    string
    bool
    float
    и другие ...

! Имена переменных должны быть Существительными
  Имена должны отражать их суть (имя - firstName, фамилия )
  Сокращение очевидных имен (msg - message)
  Использование заранее зарезервированных слов (True, bool, max()...)
  В питоне приветствуется два типа наименования переменных: CamelCase и underscore_notation
